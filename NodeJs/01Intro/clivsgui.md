CLI vs GUI
What is CLI?

CLI (Command Line Interface) is a way to interact with a system by typing commands.

Examples:

Terminal (Mac/Linux)

Command Prompt / PowerShell (Windows)

Git CLI

npm / yarn / node commands

Example:

npm install express
git commit -m "initial commit"
node server.js


You type commands ‚Üí system executes them.

What is GUI?

GUI (Graphical User Interface) lets you interact using visual elements like:

Buttons

Icons

Menus

Windows

Examples:

File Explorer

VS Code UI

Chrome browser

Postman UI

GitHub Desktop

You click things ‚Üí system performs actions.
